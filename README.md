# GUI_UNITY
Unity Workshop: Game Development Basics. Educational materials, scripts, and assets for the university seminar.

Welcome to the Unity workshop repository for the Department of Informatics - Faculty of Science - UJEP, subject GUI. The goal of this course is to understand the basics of the Unity Engine and create a functional 3D prototype within 3.5 hours.

## What are we building?
A simple 3D game where the player controls a character.

## Prerequisites (Before we start)
For a smooth workshop experience, please have the following installed:
* [**Unity Hub**](https://docs.unity3d.com/hub/manual/install-hub-win-mac.html) & **Unity 6000.2.3f1** (or newer)
* [**Visual Studio 2022/2026** (with the "Game development with Unity" workload installed)](https://visualstudio.microsoft.com/thank-you-downloading-visual-studio/?sku=Community&channel=Insiders&version=VS18&source=VSLandingPage&cid=2600&passive=false)
* Downloaded this repository -> tutorial [here](/setup_tutorials/GITHUB_ENG.md)

## Repository Structure
* `/Assets/StarterPack` – Models, textures, and sounds we will need.
* `/Assets/ExampleScripts` – Finished scripts for reference (in case you get stuck).
* `/CompletedProject` – The final version of the game for inspiration.

## Workshop Schedule
1. **Introduction to the Editor** (20 min) – Windows, navigation, game objects.
2. **Physics and Movement** (40 min) – Rigidbody, Collider, creating the first script.
3. **C# Scripting** (60 min) – Player input, variables, methods.
4. **Game Logic** (60 min) – Prefabs, triggers, simple UI.
5. **Build & Export** (30 min) – How to turn the project into an executable game (.exe).

## 💡 Useful Keyboard Shortcuts
* `W, E, R` – Move, Rotate, Scale tools.
* `F` – Focus on the selected object.
* `Ctrl + P` – Play / Pause the game.
* `Ctrl + S` – Save the scene (do this often!).
