## Jak stáhnout (naklonovat) repozitář přes GitHub Desktop

Abyste mohli s projektem pracovat u sebe v počítači, nejjednodušší je stáhnout ho pomocí aplikace GitHub Desktop. Zde je postup krok za krokem:

**Předpoklad:** Ujistěte se, že máte nainstalovanou aplikaci [GitHub Desktop](https://desktop.github.com/download/).

### Krok 1: Otevření nabídky Code
Na hlavní stránce tohoto repozitáře klikněte vpravo nahoře na zelené tlačítko **`<> Code`**.

![Krok 1 - Tlačítko Code](img/1web.png)

### Krok 2: Otevření v GitHub Desktop
V rozbalovacím menu vyberte možnost **`Open with GitHub Desktop`**. Prohlížeč se vás možná zeptá, zda chcete aplikaci povolit otevřít – potvrďte to.

![Krok 2 - Open with GitHub Desktop](img/2openGUI.png)

### Krok 3: Klonování do vašeho PC
Otevře se vám aplikace GitHub Desktop s oknem "Clone a repository". 
1. V části **Local path** klikněte na tlačítko **`Choose...`** a vyberte si složku na disku, kam chcete projekt uložit (např. `C:\UnityProjekty\GUI_UNITY`).
2. Následně klikněte na modré tlačítko **`Clone`**.

![Krok 3 - Nastavení cesty a klonování](img/3clone.png)

A to je vše! Nyní máte celý projekt stažený u sebe a můžete ho otevřít v Unity Hubu.
